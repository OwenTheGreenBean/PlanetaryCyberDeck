# PlanetWeather 🌍

A standalone cyberdeck that displays live planetary weather data using a custom-built hardware interface.

## Features

* 🌍 Planetary weather visualization
* 📡 No Wi-Fi or internet connection required
* 💾 All data is stored locally on the device
* 🖥️ Custom TFT display interface
* 🎛️ Physical controls for interacting with the display
* 🔊 Built-in speaker

## Hardware

Built around an ESP32 with a custom cyberdeck enclosure, TFT display, physical controls, and speaker.

## How It Works

Planetary weather data is downloaded and stored locally on the device, allowing the cyberdeck to operate completely offline after setup.
